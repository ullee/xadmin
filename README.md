# xadmin

# 접근제어솔루션 서버 TUI 관리툴 개발
- 접근제어솔루션 주요 데몬실행/종료, IP변경, 로깅모드 변경, DB 계정 설정 등 ADMIN 모드 적용
- 현장 지원 엔지니어들의 업무 향상에 기여 할 수 있었습니다.

> TUI 화면 예시
> ```sh
> $ make && make install
> $ ./xadmin
> ```
